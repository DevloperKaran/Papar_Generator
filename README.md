# 📝 Paper_Generator_OS
> A specialized Flask-based automation tool for structural question paper assembly.

>  This System is live now
> https://papar-generator.onrender.com

## 🚀 Project Overview
This project was designed to streamline the academic process of creating practice question papers. It allows users to browse a categorized repository of `.docx` files, select specific questions, and compile them into a new, formatted document instantly.

**Developed by:** Karan  
**Target Environment:** Maharashtra State Board / MHT CET Preparation  
**Mission:** "Syllabus Hunt" Study Group Efficiency

---

## 🛠️ Technical Stack
- **Backend:** Python 3.11 with Flask Framework
- **Frontend:** Responsive HTML5 / CSS3 (Architect Design Language)
- **Logic:** JavaScript (Asynchronous Fetch API)
- **Document Processing:** `python-docx` library
- **Hosting:** Render / GitHub Pages

---

## 📂 System Architecture
The system follows a modular folder-based hierarchy:
1. **Root Directory:** Contains `app.py` (Server logic) and `templates/` (UI).
2. **Question Bank:** Sub-folders represent subjects (e.g., Physics, Computer Science).
3. **Processing:** The server parses `.docx` files, extracts paragraphs, and maps them to unique IDs for the frontend selection.

---

## ✨ Key Features
- **Dynamic Loading:** Questions are fetched via AJAX without page reloads.
- **Architect UI:** A dark-themed, mobile-optimized interface designed for high productivity.
- **Automated Compilation:** Direct generation of `.docx` files through server-side memory buffers.
- **System Status Pulse:** Real-time feedback for loading and generation states.
